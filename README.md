# 🚀 AI Multi-Agent Development System

## 🧠 Overview
This project is an AI-driven multi-agent development system designed to automate the full software development lifecycle.

It simulates how modern AI tools collaborate to complete complex tasks including:
- Task planning
- Code generation
- Debugging & optimization

The system integrates a modular agent architecture and is designed to be extended with real LLM APIs such as GPT, Claude, and DeepSeek.

---

## ⚙️ Architecture

The system consists of three core agents:

### 1. Planner Agent
- Analyzes user requirements
- Breaks down tasks into executable steps

### 2. Coder Agent
- Generates code based on tasks
- Can be extended to call real LLM APIs

### 3. Debugger Agent
- Detects and fixes errors
- Optimizes generated code
